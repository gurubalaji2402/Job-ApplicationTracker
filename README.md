# Job Application Tracker

A full-stack web application to manage and track your job applications — built using prompt engineering with Replit Agent.

## Features

- **Dashboard** — Visual overview of your job pipeline with charts showing applications by status and recent activity stats
- **Application Management** — Add, edit, and delete job applications with details like company, role, location, salary, and notes
- **Status Tracking** — Track each application through stages: Wishlist, Applied, Interviewing, Offer, Rejected, Withdrawn
- **Search & Filter** — Search by company or role, filter by application status
- **CSV Export** — Download all your applications as a CSV file for use in Excel or Google Sheets

## How It Was Built

This entire application was created using **prompt engineering** — describing features in plain English to Replit Agent, which then wrote all the code, set up the database, and configured the project automatically.
